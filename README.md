## 美国机房建站，线路是最关键的一道门槛

搬瓦工目前美国机房主要走三类线路：

**普通线路（KVM 套餐默认）**：洛杉矶 DC2/DC3，其中 DC3 是 CN2 GT 线路。国内访问延迟在 200ms 出头，晚高峰偶尔有抖动，用来放静态博客凑合，跑电商后台就稍吃力了。

不同需求用不同的线路——这是最核心的判断逻辑。

## 搬瓦工美国 VPS 建站：从零到网站上线的步骤

这是很多人真正卡住的环节。买好 VPS 之后该怎么操作？用宝塔面板是目前新手最容易上手的路径，下面直接给步骤。

### 准备工作

在开始之前，你需要：一台搬瓦工 VPS（已购买并收到含 IP 地址的开通邮件）、一个域名（从阿里云、腾讯云、Namecheap 等地方购买均可）。

### 建站步骤

1. **登录 KiwiVM 控制面板**，确认 VPS 已启动，记录好 IP 地址和 SSH 端口号（不是默认 22，搬瓦工会给一个随机端口）

2. **用 SSH 连接 VPS**：Windows 用 PuTTY 或 Windows Terminal，macOS/Linux 直接用终端命令 `ssh root@你的IP -p 端口号`，密码来自开通邮件

3. **安装宝塔面板**，在命令行执行一行安装命令（Ubuntu/Debian 版本）：
   
   wget -O install.sh https://download.bt.cn/install/install-ubuntu_6.0.sh && sudo bash install.sh ed8484bec
   
   等待几分钟安装完成，系统会输出宝塔面板的后台地址和初始账号密码

4. **登录宝塔面板**，首次登录会提示安装 LNMP 环境（Nginx + MySQL + PHP），直接选推荐版本安装，耐心等待

5. **添加站点**：宝塔左侧点「网站」→「添加站点」，填入你的域名，选择创建 MySQL 数据库，记下数据库名称和密码

6. **解析域名**：在域名注册商的 DNS 管理界面，把你的域名 A 记录指向搬瓦工 VPS 的 IP 地址，生效时间一般 5-30 分钟

7. **安装 WordPress**：在宝塔软件商店找到「一键部署」，选 WordPress，填入域名和数据库信息，自动下载安装

8. **配置 SSL 证书**：站点设置里申请免费的 Let's Encrypt 证书，开启 HTTPS，这一步宝塔面板一键完成

安装缓存插件这件事不要省。搬瓦工 VPS 跑 WordPress，不开缓存 TTFB（首字节时间）普遍在 800ms 以上，开了之后能压到 200ms 出头，对用户体验和 Google 排名都有明显影响。
